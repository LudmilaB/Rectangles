Fit Rectangles

Fit as many rectangles as you can

0
0

HOW TO PLAY: Touch empty space to grow a new rectangle. When the new rectangle touches an existing rectangle of same color, they both disappear. If they have different colors, both stay. If the new rectangle cannot fit, a minimal dotted outline is shown. After 3 such misfits in a row, the game is over. If you touch inside a standalone rectangle, it will grow more.
